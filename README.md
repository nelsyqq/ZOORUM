# ЗООРУМ — Зоомагазин

Многостраничный веб-сайт зоомагазина на Vue 3 с современным минималистичным дизайном.

## Стек

- **Vue 3** (Composition API)
- **Vite**
- **Vue Router 4**
- **Pinia**
- **Tailwind CSS**
- **Lucide Vue** (иконки)

## Запуск

### Способ 1 — двойной клик (Windows)

Дважды нажмите на файл **`start.bat`** в папке проекта.  
Сайт откроется в браузере автоматически.

### Способ 2 — через терминал

```bash
npm install
npm run dev
```

Сайт откроется на **http://localhost:5173**

> **Важно:** не открывайте `index.html` напрямую в браузере — Vue-приложение работает только через dev-сервер (`npm run dev`) или после сборки (`npm run build` + `npm run preview`).

## Функционал

- Каталог товаров с фильтрацией и поиском
- Корзина (Pinia) с оформлением заказа
- Форма обратной связи на всех страницах (модальное окно)
- Личный кабинет: вход, регистрация, профиль, история заказов
- Админ-панель: CRUD товаров, управление заказами и пользователями
- Мобильная адаптивность (mobile-first)

## Работа в России

- **Шрифты** встроены в проект (`@fontsource`) — Google Fonts не нужен
- **Картинки** лежат локально в `public/images/` — без Unsplash и CDN
- **Данные** хранятся в `localStorage` браузера (корзина, вход, заказы)

## Как проверить корзину

1. Добавьте товары в каталоге → откроется корзина
2. Нажмите «Оформить заказ» → укажите адрес
3. Если не вошли — перенаправит на вход, после входа корзина откроется снова
4. Заказ появится в **Личный кабинет → Заказы**
5. Обновите страницу — корзина и заказы сохранятся
