# ЗООРУМ — Зоомагазин

Многостраничный веб-сайт зоомагазина на Vue 3. Каталог, корзина, админ-панель, отзывы, синхронизация между устройствами через Cloudflare KV.

## Стек

- **Vue 3** (Composition API, `<script setup>`)
- **Vite**
- **Vue Router 4**
- **Pinia** (стейт-менеджмент)
- **Tailwind CSS**
- **Lucide Vue** (иконки)
- **Cloudflare Pages** (хостинг + KV для кросс-девайс синхронизации)

## Запуск

```bash
npm install
npm run dev
```

Сайт откроется на **http://localhost:5173**

## Сборка / Деплой

```bash
npm run build         # сборка в dist/
git push origin main  # авто-деплой на Cloudflare Pages
```

Сайт: **https://zoorum.pages.dev**

## Страницы

| Маршрут | Описание |
|---------|----------|
| `/` | Главная с товарами по категориям |
| `/catalog` | Каталог с фильтрацией и поиском |
| `/catalog/:id` | Карточка товара (галерея, отзывы) |
| `/about` | О нас |
| `/contacts` | Контакты |
| `/profile` | Личный кабинет (вход/регистрация/профиль/заказы) |
| `/admin` | Админ-панель (7 вкладок) |

## Функционал

- Каталог товаров с фильтрацией по категориям и поиском
- Автодополнение поиска (по первому символу)
- Мульти-изображения товаров (галерея с превью)
- Варианты веса/фасовки с разными ценами и остатками
- Старая цена (crossed-out price) и скидки
- Корзина с оформлением заказа
- Отзывы на товары: рейтинг, текст, аватар автора, редактирование/удаление в течение 30 дней
- Форма обратной связи (модальное окно на всех страницах)
- Личный кабинет: вход/регистрация, редактирование профиля, загрузка аватара, история заказов
- Админ-панель: CRUD товаров/категорий, управление заказами/пользователями/отзывами, просмотр обратной связи
- Экспорт/импорт всех данных (JSON)
- Кросс-девайс синхронизация через Cloudflare KV + localStorage (fallback)
- Авто-синхронизация каждые 20 секунд
- Соц-сети в футере: Telegram, VK, WhatsApp
- Адаптивность (mobile-first)
- Плавные анимации и переходы между страницами

## Изображения

Все изображения — локальные JPG/PNG в `public/images/`:
- `products/` — товары (в т.ч. загруженные через админку)
- `categories/` — категории
- `hero/` — главный экран
- `logo.jpg`, `placeholder.jpg` и др.

Загрузка через админ-панель конвертирует файлы в base64 и сохраняет в данных. Для переноса на GitHub используется скрипт `scripts/save-images.cjs`.

## Работа в России

- Шрифты встроены (`@fontsource`) — Google Fonts не требуется
- Изображения локальные — без внешних CDN
- Данные хранятся в localStorage + Cloudflare KV
